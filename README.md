# Hospital Readmission Prediction Model

This project predicts whether a hospital patient will be readmitted using a Machine Learning model.

## Features Used
- Age
- Gender
- Diagnosis
- Treatment
- Days in Hospital
- Number of Previous Admissions

## Model
Random Forest Classifier

## Technologies
- Python
- Pandas
- Scikit-learn
- Google Colab

## Files
hospital_llm_model.ipynb → ML model code  
hospital_patients.csv → Dataset
